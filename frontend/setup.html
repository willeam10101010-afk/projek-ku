Step 1: Install Dependencies

Make sure you have Node.js installed (v16 or higher), then run:

npm install

Step 2: Configure Environment

Copy the example environment file:

cp .env.example .env

Edit .env and add your configuration:

  • Private key from your wallet (for deployment)
  • RPC URL for your chosen network

Step 3: Get Testnet ETH

Get free testnet ETH from these faucets:

  • Sepolia Faucet
  • Goerli Faucet

Step 4: Compile Contracts

npm run compile

This will compile the Solidity smart contracts.

Step 5: Run Tests

npm test

Ensure all tests pass before deployment.

Step 6: Deploy to Testnet

npm run deploy:testnet

This will deploy the contracts to Sepolia testnet.

Save the contract addresses from the output!

Step 7: Update Frontend Configuration

Edit frontend/app.js and update:

const MINING_CONTRACT_ADDRESS = "0x...";
const USDT_CONTRACT_ADDRESS = "0x...";

Step 8: Start the DApp

In one terminal, start the frontend:

npm run start:frontend

In another terminal, start the backend:

npm run start:backend

Visit http://localhost:8080 in your browser.
